neogcamp : how I learnt to make this website

What is neogcamp?

A bootcamp that revives the learning for new(neo) generation(+G) programmers.

The science part of computer science engineering has become about mugging up Data Structures and Algorithms. Neogcamp aims to bring back the joy of creating, the engineering part. Do more, create more, learn while doing.

Your guide for this program is Tanay Pratap. I liked looking at how interactive and beautiful different websites were, so when I discovered about a course on web development by Neogcamp, I knew I had to try. Through out this course, I have studied about the basics of HTML, CSS, Javascript, Git and React. This beginner-friendly course has taught me the basics and now I am learning about the advanced nuances of web development.
